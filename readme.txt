Задание 24 ( 4 ). Задается словарь. Найти в нем все анаграммы (слова, составленные из одних и тех же букв).
Запуск программы осуществлять при помощи консоли.
Ввод осуществляется с файла file.txt.
Слова в файле file.txt расположены на каждой строчке отдельно ( т.е. одно слово - одна строка ).
Тестировалось на Windows 10.
